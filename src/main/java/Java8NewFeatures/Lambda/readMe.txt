
使用Annotation,书写完成TestCase,要求如下：
（1）定义一个单值注解TestCase，使其可以注解方法，并且可以被保留到程序运行时注解的属性类型为String，要求可以使用简写方式为属性赋值
（2）定义一个类TestCaseDemo，
要求有三个方法methodA、methodB、methodC方法的参数、返回值类型均为String类型，参数名称均定义为name,返回值为传入的参数,方法体中逻辑均简单实现为传入字符串追加任意字符串
使用（1）中的注解来注释三个方法中的一个,并对属性参数赋值
（3）定义一个测试类TaseCaseTest，要求使用反射来测试TestCaseDemo中所有的被TestCase注解的方法
并将注解的属性值作为参数，调用相应方法来返回测试结果
